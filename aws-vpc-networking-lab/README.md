# AWS VPC Networking Lab

A hands-on AWS lab covering core VPC networking concepts, including VPC creation, traffic flow, private subnets, connectivity testing, peering, flow logs, S3 access, and endpoints.

## Structure

- `assets/architecture-diagrams/` — reference network diagrams
- `assets/screenshots/` — lab screenshot folders
- `01-build-a-virtual-private-cloud/` — step-by-step VPC creation
- `02-vpc-traffic-flow-and-security/` — traffic flow and security controls
- `03-creating-a-private-subnet/` — private subnet configuration
- `04-testing-vpc-connectivity/` — connectivity testing scenarios
- `05-vpc-peering/` — peering connections and routing
- `06-vpc-monitoring-with-flow-logs/` — VPC flow log setup and insights
- `07-access-s3-from-a-vpc/` — private S3 access from within a VPC
- `08-vpc-endpoints/` — VPC endpoint usage and configuration
